# Just tweet and code

## Installation

- Create the `secret.py` in `./src`, then insert your consumer and access token key.
- Run `pip install .`

## Usage

`tw [the message]`

## Limitation
Now, the module cannot handle some cli-syntax symbol such as `>`. So, please use this module wisely.
